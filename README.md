## English

### 📋 Project Overview

This project is an automated API testing framework using **Java**, **Cucumber (BDD)**, and **RestAssured**. It validates the functionalities of the **Serverest API**, covering login and user management processes (CRUD).

### ⚙️ Installation

```bash
git clone https://github.com/yourusername/yourproject.git
cd yourproject
mvn clean install
```

### ▶️ Running Tests

```bash
mvn clean test && mvn cluecumber-report:reporting
```

## Português

### 📋 Visão Geral do Projeto

Este projeto é um framework de testes automatizados de API utilizando **Java**, **Cucumber (BDD)** e **RestAssured**.

### 🚀 Pré-requisitos

- JDK 8 ou superior
- Maven instalado
- IDE (IntelliJ, Eclipse, etc.)

### ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/yourusername/yourproject.git
```

Acesse a pasta do projeto:

```bash
cd yourproject
```

Instale as dependências:

```bash
mvn clean install
```

### ▶️ Executando os testes

```bash
mvn clean test && mvn cluecumber-report:reporting
```
